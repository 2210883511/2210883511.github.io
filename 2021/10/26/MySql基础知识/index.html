MySql基础知识

  1. MySql常见命令

    • 查看当前所有的数据库

      show databases;

    • 打开指定的库

      use + 库名

    • 查看当前库的所有表

      show tables;

    • 查看其它库的所有表

      show tables from + 库名;

    • 创建表

    • 查看表结构

      desc + 表名;

    • 插入语句

      insert into + 表名 + (字段名1，字段名2，...) values(val_1,val_2,...);

    • 查询语句

      select * from + 表名 (查询全部)

    • 更新语句

      update + 表名 + set +字段名 = val where 字段名 = val

    • 删除语句

      delete from + 表名+where + 字段名 = val

    • 查看服务器的版本

      select version(); 登录到mysql服务端

      mysql -v;没有登录的mysql服务端
